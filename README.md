# Instalação
 <h3>npm install, to install all dependencies</h3>
 <h3>npm run dev, to run</h3> 


# Configuration
  Gere sua Api key => https://home.openweathermap.org/api_keys
  substitua a na pasta services => WeatherApi.jsx => API_KEY.
  Não altere a API_KEY2
